## 2. Core Services
Aavarna Spaces specializes in both structural and aesthetic property development:
1. **Construction Services:** Robust and reliable structural execution from foundation to roof, ensuring lifetime durability with advanced engineering precision.
2. **Interior Design:** Aesthetic, functional, and personalized interior solutions that transform generic rooms into curated living and working spaces.
3. **Renovation & Remodeling:** Breathing new life into aging structures with comprehensive modern remodeling services designed to enhance property value.
4. **Project Management:** Seamless coordination between stakeholders, timely execution, and rigorous quality control for the client's absolute peace of mind.
5. **Commercial Spaces:** Optimized workspaces, corporate HQs and retail environments designed for productivity, flow, and modern brand identity.
6. **Residential Projects:** Turnkey custom homes tailored to the client's lifestyle, delivering safety, maximum comfort, and striking architectural beauty.

## 3. The "Aavarna Process"
A streamlined 4-step execution strategy mapping the customer journey:
1. **Consultation:** Deep-diving into the client's vision, requirements, lifestyle needs, and overall project scope.
2. **Design:** Creating comprehensive architectural plans, 3D views, and curating premium material selections.
3. **Execution:** On-site construction and precision interior work managed with strict quality benchmarks.
4. **Handover:** Final detailing, deep cleaning, and the official delivery of keys to the immaculate new space.

## 8. Website Tech Stack & Structure
The official website is a high-performance static site built to emphasize aesthetics without relying on JavaScript or external libraries (other than Google Fonts):
- **Core Technology:** HTML5, CSS3 (No JavaScript framework used; pure CSS logic and pseudo-classes for interactivity).
- **Styling Paradigm:** Custom Vanilla CSS utilizing CSS Variables (`:root`), Flexbox, CSS Grid, and responsive Media Queries.
- **Typography:** Google Font - 'Outfit' (Weights: 300, 400, 600, 800) for a modern, geometric tech-oriented yet elegant feel.
- **Color Palette:**
  - *Primary (Deep/Authority):* Dark Blue (`#0d1b2a`) and Secondary Navy (`#1b263b`)
  - *Accent (Luxury/Action):* Premium Gold (`#d4af37`), hover state (`#b5952f`)
  - *Backgrounds (Cleanliness):* Crisp White (`#ffffff`), Soft Off-White (`#f4f4f4`)
  - *Text (Readability):* Dark Gray (`#333333`), Muted Gray (`#6c757d`), Light/White text on dark backgrounds (`#f8f9fa`)
- **Animations:** CSS Keyframe animations (`fadeInDown`, `fadeInUp`) for hero section load-in, CSS transitions for hover states (buttons, cards, images scaling).

### Component Layout:
1. **Header:** Sticky-top blurred navbar.
2. **Hero Section (`#hero`):** Full-viewport landing with strong H1 and dual-toned gradient background over architecture image.
3. **Services (`#services`):** CSS Grid cards with hover effect expanding background fill. Utilizes HTML entity fallbacks as icons, paired with background images.
4. **About (`#about`):** Split layout (Image + Text) with a modern overlapping "10+ Years" experience badge.
5. **Projects (`#projects`):** Image gallery utilizing CSS Grid. Hovering scales the image and summons a data overlay.
6. **Process (`#process`):** Pseudo-element connected step track using circles.
7. **Testimonials (`#testimonials`):** Elegant typographic quotes with large opaque serif quote marks for styling.
8. **Contact (`#contact`):** Dual split form; left side holds info, right side holds a native HTML form with CSS validation styling. Includes dummy `.preventdefault()` functionality.
9. **Footer:** Standard 3-column link categorization and copyright footer.
